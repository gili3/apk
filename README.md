## 🌟 Overview
This is a fully native Android application for **Eleven Store**, rebuilt from the ground up using Kotlin and Jetpack Compose. It features a modern MVVM architecture, Firebase integration, and a seamless user experience.

## 🚀 Features
- **Native Performance**: 100% Kotlin Native, no WebViews.
- **Modern UI**: Built with Jetpack Compose for smooth animations and RTL support.
- **Firebase Integrated**: Auth, Firestore, Storage, and FCM.
- **CI/CD Ready**: GitHub Actions workflow included for automated APK/AAB builds.

## 🛠️ Build Instructions
1. Clone this repository.
2. Open in **Android Studio Hedgehog** or newer.
3. Sync Gradle.
4. Use the provided `release-keystore.jks` for signing.
